How to use a NTUC Fairprice voucher code?
  1. Access the e-voucher by clicking on the URL
  2. E-voucher will load on a browser (not required to download the app)
  3. E-voucher is ready for use in stores or online. E-voucher status and QR code will be reflected
Note: Further instructions on how to use the e-voucher and terms & conditions can be found upon clicking on the URL

Redeem in stores:
At Cashier
  1. Show the QR Code to the cashier
At Self-Checkout Counter
When selecting payment type:
  1. Select All Other Payment
  2. Select Partner Voucher/ FP eVoucher
  3. Use handheld scanner to scan QR code

 

Redeem Online:
  1. Tap on “Copy” to copy the voucher code
  2. Shop online at fairprice.com.sg OR the FairPrice App
  3. Tap on “Add” voucher code at checkout
  4. Paste the code you copied, tap “Apply” and proceed with checkout
